# 🍏 Nutri-App  

## 📖 Descripción  
NutriConsciente es una aplicación enfocada en la alimentación consciente y el bienestar integral. Permite a los usuarios:  
- Registrar sus comidas diarias.  
- Recibir y personalizar recetas saludables.  
- Contactar especialistas en nutrición y psicología.  
- Acceder a contenido educativo y mensajes motivacionales.  

## 🎯 Objetivo  
Facilitar el cumplimiento de metas alimenticias, fomentar la educación en nutrición y ofrecer un canal directo de ayuda profesional.
